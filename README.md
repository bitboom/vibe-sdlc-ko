# Vibe Coding 시대의 새로운 SDLC — Korean MVP

Kaggle whitepaper **The New SDLC With Vibe Coding**를 한국어로 구조화한 GitHub Pages용 1차 MVP입니다.

원문: https://www.kaggle.com/whitepaper-the-new-SDLC-with-vibe-coding

## 목표

- 원문의 핵심 의도를 보존한 한국어 요약/주요 섹션 번역
- 개발자와 엔지니어링 리더가 읽기 좋은 문서형 UX
- GitHub Pages에서 바로 호스팅 가능한 no-build 정적 사이트

## 1차 MVP 범위

- Executive Summary
- Vibe Coding vs Agentic Engineering
- Context Engineering: 6가지 context, static/dynamic context, Agent Skills
- 새로운 SDLC: 요구사항, 설계, 구현, 테스트/QA, 리뷰/배포, 유지보수
- Factory Model
- Harness Engineering: Agent = Model + Harness, harness 구성 요소, SDLC 적용
- 개발자 역할 변화: Conductor/Orchestrator, The 80% Problem
- AI 개발의 경제학: TCO, CapEx/OpEx, token burn rate, maintenance/security cost, model routing
- 시작 가이드: 개인 개발자, 엔지니어링 리더, 조직별 적용 단계
- UX polish: 섹션별 takeaway box, 모바일 목차 drawer
- SEO/OG: canonical URL, Open Graph/Twitter card, 공유용 OG 이미지, favicon
- 용어집

전체 51페이지 완역은 후속 단계에서 섹션별로 확장합니다. 현재는 핵심 섹션을 원문 기반으로 구조화한 reading guide입니다.
